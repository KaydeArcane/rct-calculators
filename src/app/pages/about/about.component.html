RCT Calculators

A collection of useful calculators for use with OpenRCT2, Roller Coaster Tycoon 1, Roller Coaster Tycoon 2, and Roller Coaster Tycoon Classic.

Developed by @KaydeArcane . Data sourced from OpenRCT2 repo. Ride price, shop price, and soft guest cap calculations based on discoveries by Deurklink.

Massive shout out to Marcel Vos for covering these mechanics on YouTube and spurring my interest in making these tools!

FAQs

Where is [insert attraction] in the dropdown list?

The dropdown lists should cover every attraction possible to build in OpenRCT2 and RCT Classic. However, some rides are reskins of a base ride type - for example, the "Snow Cups" ride is actually a reskin of the "Twist" ride. The calculations for a reskinned ride is the exact same as the ride it's based on, so if you are using a reskinned ride, add the base ride type instead.

If you think an attraction is missing, file a bug report on the Github repo with detailed information about the missing ride.

Why did my ride get marked as a duplicate of another ride of a different type?

Some rides are considered the same type for the game's ride value and price calculation mechanics, despite not being exact reskins of other rides. These rides are as follows:

  • Car Ride / Monster Trucks
  • Classic Mini Roller Coaster / Junior Roller Coaster
  • Steel Wild Mouse / Spinning Wild Mouse
  • Hyper-Twister Coaster / Steel Twister Coaster
  • Corkscrew Roller Coaster / Hypercoaster

How do I tell how old my ride is? The game only specifies the year it was built in.

This is tricky, because as far as I can find, there is no way in game to see exactly how many months old a ride is. The best way I've found to deal with this is to keep track of guests thoughts to see when they say they won't pay that much to ride a given ride, and increase the age value when they do to get the new admission price.

How can I find out if a scenario I'm playing has harder guest generation enabled?

The official list of scenarios that use harder guest generation is as follows:

  • Alpine Adventures (RCT2)
  • Frightmare Hills (RCT: Loopy Landscapes)
  • Nevermore Park (RCT: Loopy Landscapes)
  • Southern Sands (RCT: Loopy Landscapes)
  • Stone Age (RCT2: Time Twister)
  • Venus Ponds (RCT: Loopy Landscapes)

If you're playing on OpenRCT2, you can also check by opening the cheats menu and selecting "Scenario options", then looking at the park options tab. If the "Guests are more difficult to attract" setting is checked, the scenario has harder guest generation enabled.
